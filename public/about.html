How It Works

From raw text to structured trees in three steps.

1. Define Relationships

Type or click to create edges like A→B. Each edge says "A is the parent of B." You can add as many as you want — we'll sort it out.

2. Automatic Validation

When you hit Process, Nodetree runs your edges through a pipeline:

  • Format check — only [A-Z]→[A-Z] edges pass validation
  • Deduplication — repeated edges are kept once, extras are flagged
  • Single parent rule — each node can only have one parent
  • Cycle detection — circular references are identified using DFS

3. Visualize Results

Valid edges produce an interactive tree view. You'll see depth, node counts, root identification, and any issues — all at a glance. Export as JSON or share the link.
